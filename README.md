# iHoover
* Mini projet / Test Technique Yanport | **Angular TypeScript**

# Installation

* Clone the repositry
```bash
git clone https://github.com/0Stain/iHoover-App
```

* Installer les dependeces

```bash
npm install
```

# Lancer le projet

```bash
npm start
```

Trouvez l'appli sur :
**https://localhost:4200/**
